1. Getting started with pyvisdk¶

This will be uploaded to the Python Package Index when it becomes more stable, but for now you can download the code from github, then run the following:

$ python ./bootstrap.py $ ./bin/buildout

To install, run the following:

$ sudo python setup.py

1.1. Requirements¶

  • dataflake.cache
  • suds
