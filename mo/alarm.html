ReconfigureAlarm(spec)[source]¶

Reconfigures the alarm properties. This operation requires access privileges on the entity with which the alarm is associated.In addition to the Alarm.Edit privilege, may also require the Global.ScriptAction if a RunScriptAction action is specified in the AlarmSpec.

Parameters:spec – The new specification for the alarm.
RemoveAlarm()[source]¶

List of custom field definitions that are valid for the object’s type. The fields are sorted by name.

Information about this alarm.

value¶

List of custom field values. Each value uses a key to associate an instance of a CustomFieldStringValue with a custom field definition.
