Properties

CheckCompliance_Task(profile, entity)[source]¶

Check compliance of an entity against a Profile.

Parameters:profile – If specified, check compliance against the specified profiles. If not specified,

use the profiles associated with the entities. If both Profiles and Entities are specified, Check the compliance of each Entity against each of the profile specified. For more information, look at the KMap below. P represents if Profile is specified. E represents if Entity is specified.

Parameters:entity – If specified, the compliance check is done against this entity.
Return type:Task
ClearComplianceStatus(profile, entity)[source]¶

Clear the saved ComplianceResult based on profile and entity filtering criteria.

Parameters:
  • profile – If specified, clear the ComplianceResult related to the Profile.
  • entity – If specified, clear the ComplianceResult related to the entity. If profile and

entity are not specified, all the ComplianceResults will be cleared.

QueryComplianceStatus(profile, entity)[source]¶

Query the compliance status based on Profile and Entity filter.

Parameters:profile – If specified, compliance result for the specified profiles will be returned. This

acts like a filtering criteria for the ComplianceResults based on specified profiles.

Parameters:entity – If specified, compliance results for these entities will be returned. This acts

like a filtering criteria for the ComplianceResults based on entities.

Return type:ComplianceResult[]
QueryExpressionMetadata(expressionName)[source]¶

Query the metadata for the expressions.

Parameters:expressionName – Names of the Expressions for which metadata is requested. If expressionNames are

not specified, metadata for all known expressions is returned

Return type:ProfileExpressionMetadata[]
