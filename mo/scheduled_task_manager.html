CreateScheduledTask(entity, spec)[source]¶

Creates a scheduled task.

Parameters:entity – The managed entity (or entities) for which the scheduled task triggers an action.

You can schedule tasks on any managed entity. If the scheduled task is associated with a leaf node in the inventory tree, it applies only to a single entity (virtual machine or host). If the task is associated with a folder, a datacenter, a compute resource, or a resource pool, it applies to the virtual machine or host descendants of the entity.

Parameters:spec – The specification for the new scheduled task.
Return type:ScheduledTask
