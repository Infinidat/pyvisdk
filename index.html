pyvmware

Welcome

pyvisdk is a pure python library to access the VMware vSphere web services API. Thanks to psphere for giving me some insight into how the API works.

Features

  • Pure Python implementation.
  • A more Object Oriented approach to interfacing with the web services API.
  • Knowledge of the inner workings of the API is not needed.
  • Simple and clean interface.

Get pyvisdk

pyvisdk is available as an easy-installable package on the Python Package Index.

The code can be found at github, at https://github.com/xuru/pyvisdk.

Download

Current version: 0.7

Get pyvisdk from the Python Package Index, or install it with: easy_install pyvisdk
