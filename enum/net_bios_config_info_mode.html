7.103. NetBIOSConfigInfoMode¶

NetBIOSConfigInfoMode
NetBIOS configuration mode.
NetBIOSConfigInfoMode.disabled¶
NetBIOS is disabled.
NetBIOSConfigInfoMode.enabled¶
NetBIOS is enabled.
NetBIOSConfigInfoMode.enabledViaDHCP¶
DHCP server decides whether or not to use NetBIOS.
NetBIOSConfigInfoMode.unknown¶
Mode of NetBIOS is unknown.
