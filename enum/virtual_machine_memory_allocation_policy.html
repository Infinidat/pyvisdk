7.170. VirtualMachineMemoryAllocationPolicy¶

VirtualMachineMemoryAllocationPolicy

Means for allocating additional memory for virtual machines.

VirtualMachineMemoryAllocationPolicy.swapMost¶

Allow most virtual machine memory to be swapped.

VirtualMachineMemoryAllocationPolicy.swapNone¶

Fit all virtual machine memory into reserved host memory.

VirtualMachineMemoryAllocationPolicy.swapSome¶

Allow some virtual machine memory to be swapped.
