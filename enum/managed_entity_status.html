7.100. ManagedEntityStatus¶

ManagedEntityStatus
The Status enumeration defines a general "health" value for a managed entity.
ManagedEntityStatus.gray¶
The status is unknown.
ManagedEntityStatus.green¶
The entity is OK.
ManagedEntityStatus.red¶
The entity definitely has a problem.
ManagedEntityStatus.yellow¶
The entity might have a problem.
