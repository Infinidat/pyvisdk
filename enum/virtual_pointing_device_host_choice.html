7.188. VirtualPointingDeviceHostChoice¶

VirtualPointingDeviceHostChoice
The valid choices for host pointing devices are:
VirtualPointingDeviceHostChoice.autodetect¶
Automatically detects the host mouse type.
VirtualPointingDeviceHostChoice.intellimouseExplorer¶
The Microsoft IntelliMouse Explorer.
VirtualPointingDeviceHostChoice.intellimousePs2¶
The Microsoft Intellimouse with a PS2 connection.
VirtualPointingDeviceHostChoice.logitechMouseman¶
The Logitech MouseMan.
VirtualPointingDeviceHostChoice.microsoft_serial¶
The Microsoft Serial Mouse.
VirtualPointingDeviceHostChoice.mousemanSerial¶
The Logitech MouseMan Serial Bus Mouse.
VirtualPointingDeviceHostChoice.mouseSystems¶
The Mouse Systems Mouse.
VirtualPointingDeviceHostChoice.ps2¶
A generic mouse with a PS2 connection.
