7.122. RecommendationReasonCode¶

RecommendationReasonCode
List of defined migration reason codes:
RecommendationReasonCode.antiAffin¶
Fulfill anti-affinity rule.
RecommendationReasonCode.checkResource¶
Sanity-check resource pool hierarchy vSphere API 4.0
RecommendationReasonCode.enterStandby¶
Host entering standby mode.
RecommendationReasonCode.fairnessCpuAvg¶
Balance average CPU utilization.
RecommendationReasonCode.fairnessMemAvg¶
Balance average memory utilization.
RecommendationReasonCode.hostMaint¶
Host entering maintenance mode.
RecommendationReasonCode.increaseCapacity¶
Power on host to increase cluster capacity
RecommendationReasonCode.jointAffin¶
Fulfill affinity rule.
RecommendationReasonCode.powerOnVm¶
Power on virtual machine
RecommendationReasonCode.powerSaving¶
Power off host for power savings
RecommendationReasonCode.reservationCpu¶
balance CPU reservations
RecommendationReasonCode.reservationMem¶
balance memory reservations
RecommendationReasonCode.unreservedCapacity¶
Maintain unreserved capacity vSphere API 4.0
RecommendationReasonCode.vmHostHardAffinity¶
Fix hard VM/host affinity rule violation vSphere API 4.1
RecommendationReasonCode.vmHostSoftAffinity¶
Fix soft VM/host affinity rule violation vSphere API 4.1
