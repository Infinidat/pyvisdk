7.115. PerformanceManagerUnit¶

PerformanceManagerUnit

Indicates the unit of measure represented by a counter or statistical value.

PerformanceManagerUnit.joule¶

Joules vSphere API 4.1

PerformanceManagerUnit.kiloBytes¶

Kilobytes.

PerformanceManagerUnit.kiloBytesPerSecond¶

Kilobytes per second.

PerformanceManagerUnit.megaBytes¶

Megabytes.

PerformanceManagerUnit.megaBytesPerSecond¶

Megabytes per second.

PerformanceManagerUnit.megaHertz¶

Megahertz.

PerformanceManagerUnit.microsecond¶

The time in microseconds. vSphere API 4.0

PerformanceManagerUnit.millisecond¶

The time in milliseconds.

PerformanceManagerUnit.number¶

A quantity of items, for example, the number of CPUs.

PerformanceManagerUnit.percent¶

Percentage.

PerformanceManagerUnit.second¶

The time in seconds.

PerformanceManagerUnit.watt¶

Watts vSphere API 4.1
