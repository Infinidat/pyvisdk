
==================================================================================================
PhysicalNicResourcePoolSchedulerDisallowedReason
==================================================================================================

.. describe:: PhysicalNicResourcePoolSchedulerDisallowedReason

    

    
    .. py:data:: PhysicalNicResourcePoolSchedulerDisallowedReason.hardwareUnsupported

        Indicates that the NIC device does is not capable of resource pool based scheduling.

    
    .. py:data:: PhysicalNicResourcePoolSchedulerDisallowedReason.userOptOut

        Indicates that the user has opted out the Physical NIC from resource pool based scheduling.
