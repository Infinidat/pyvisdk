
==================================================================================================
VasaProviderStatus
==================================================================================================

.. describe:: VasaProviderStatus

    The operational state of VASA Provider.

    
    .. py:data:: VasaProviderStatus.offline

        VASA Provider is offline.

    
    .. py:data:: VasaProviderStatus.online

        VASA Provider is operating correctly.

    
    .. py:data:: VasaProviderStatus.syncError

        VASA Provider is reachable, but sync operation failed.

    
    .. py:data:: VasaProviderStatus.unknown

        VASA Provider is unreachable.
