
==================================================================================================
NodeStatus
==================================================================================================

.. describe:: NodeStatus

    Values to convey information about the functional status of a node in the topology. up - indicates that the entity is fully operational. degraded - indicates that there are potential problems with the entity and/or its sub-components or that it is operating in a reduced/degraded state. down - indicates a critical failure. unknown - indicates that the status is not known. notReported - indicates that the status is not reported.
