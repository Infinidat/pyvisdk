
==================================================================================================
VirtualMachineFlagInfoMonitorType
==================================================================================================

.. describe:: VirtualMachineFlagInfoMonitorType

    Set of possible values for monitorType.

    
    .. py:data:: VirtualMachineFlagInfoMonitorType.debug

        Run vmx in debug mode.

    
    .. py:data:: VirtualMachineFlagInfoMonitorType.release

        Run vmx in default mode, matching the build type of vmkernel.

    
    .. py:data:: VirtualMachineFlagInfoMonitorType.stats

        Run vmx in stats mode.
