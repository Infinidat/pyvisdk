6.184. DailyTaskScheduler¶

Extended by

MonthlyTaskScheduler, WeeklyTaskScheduler

Extends

HourlyTaskScheduler

class pyvisdk.do.daily_task_scheduler.DailyTaskScheduler¶
hour¶

The hour at which the RecurrentTaskScheduler runs the task. Use UTC (Coordinated Universal Time) values in the range 0 to 23, where 0 = 12:00 a.m. (UTC) and 12 = 12:00 p.m. (UTC).
