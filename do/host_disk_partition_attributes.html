4.496. HostDiskPartitionAttributes¶

Property of

HostDiskPartitionSpec

Extends

DynamicData

class pyvisdk.do.host_disk_partition_attributes.HostDiskPartitionAttributes¶
attributes¶

The attributes on the partition.

endSector¶

The end sector.

logical¶

The flag to indicate whether or not the partition is logical. If true, the partition number should be greater than 4.

partition¶

The partition number. Must be a positive integer.

startSector¶

The start sector.

type¶

Type of data in the partition. If it is a well-known partition type, it will be one of the defined types. If it is not, then it will be reported as a hexadecimal number. For example, “none”, “vmfs”, “linux”, and “0x20” are all valid values.
