6.1157. VirtualMachineDatastoreVolumeOption¶

Property of

DatastoreOption

Extends

DynamicData

class pyvisdk.do.virtual_machine_datastore_volume_option.VirtualMachineDatastoreVolumeOption¶
fileSystemType¶

The type name of the file system volume information object for this option.See HostFileSystemVolumeInfo

majorVersion¶

The major version of the file system volume information for this option. If not specified, all versions of this file system are included in this option. Currently, this value is set only for VMFS volumes.
